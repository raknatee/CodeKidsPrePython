#### Naming Dos and Don'ts
- Start with a letter
- Symbols and spaces are not allowed (-, /, #, @) 
- The underscore can be used (For example, first_name, last_name)

### Strings

#### Assignment
```python
my_name = "Ally Alien"
print(my_name)
```
::: details output
```
Ally Alien
```
:::

####  Combination
```python
first_name = "Ally"
last_name = "Alien"
full_name = first_name + last_name
print(full_name)
```
::: details output
```
AllyAlien
```
:::
```python

```
::: details output
```

```
:::
