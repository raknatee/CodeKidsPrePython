# How to get started

- VuePress Guide https://vuepress.vuejs.org/

## Run local server
- PS. install Docker first

```sh
docker-compose up --build -d
```
```sh
docker-compose exec project bash
```
```sh
yarn dev
```

## Run deploy Github Page

```sh
docker-compose up --build -d
```
```sh
docker-compose exec yarn build
```
